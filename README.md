# Metadata Analyzer

A modern web application that allows users to upload images and view their metadata. Built with React, TypeScript, and Material-UI.

## Features

- Drag and drop image upload
- Image preview
- Metadata extraction and display
- Dark mode support
- Responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`
